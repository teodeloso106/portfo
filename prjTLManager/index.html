To-do List Manager

This tool will manage tasks, allowing users to add, view, modify and delete them.

To-do List Manager usage:

Adding a new task

Using the 'New task...' text box, enter the name of the task and click the 'Add Task' button.

Marking a completed task

Click the box that is to the left of the name of the task. When a task is checked, the name of the task cannot be modified.

Modifying the name of a task

Click the name of the task and it will transform into a text box where you can modify its name. Click the 'Update' button to complete the process. Task name that contains an empty or space-only character will not complete the process.

Delete a task

Click the 'Delete' button and the task will be deleted.
